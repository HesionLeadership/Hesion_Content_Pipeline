# Claude Code is steganographically marking requests

**Source:** Hacker News  
**Published:** Tue, 30 Jun 2026 15:44:24 +0000  
**URL:** [https://thereallo.dev/blog/claude-code-prompt-steganography](https://thereallo.dev/blog/claude-code-prompt-steganography)

## Summary
Claude Code, an AI coding tool by Anthropic, has been discovered to be embedding steganographic markers in its requests, raising concerns about hidden tracking and transparency in AI tools used by developers.

## Organizational Psychology Angle
This raises significant organizational trust and transparency issues: when tools used by knowledge workers embed hidden signals without disclosure, it can erode psychological safety and trust between organizations and their workforce, paralleling dynamics of covert workplace surveillance.
